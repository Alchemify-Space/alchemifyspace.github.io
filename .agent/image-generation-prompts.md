## IMAGE GENERATION TOOLS RECOMMENDED

1. **Midjourney** - Best for creative, artistic images with cosmic/tech themes
2. **DALL-E 3** - Great for precise, descriptive prompts
3. **Stable Diffusion** - Free/open source option
4. **Canva AI** - Easy to use for consistent branding
5. **Adobe Firefly** - Professional quality, commercially safe

## CONSISTENCY TIPS

- Always include brand color #2A9D8F (teal) in prompts
- Use keywords: "modern", "clean", "professional", "tech", "minimalist"
- Maintain consistent style across similar images
- Use "flat design" for icons and illustrations
- Use "product photography" for mockups and interfaces
- Add "cosmic", "nebula", "transformation" for brand-themed imagery
